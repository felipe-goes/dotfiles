# Shortcuts for vim-fugitive

- <leader>gg  :G - Open the vim-fugitive interface to make Git operations.
- :Git push      - Push changes. No shortcut for this one yet.

## Shortcuts once already in the vim-fugitive interface

- dd - git diff.
- dq - close git diff.
- cc - commit.
- s  - stage.
- u  - unstage.
